# 이메일 자동화
- 대량 발송시에는 대량 발송 서비스 이용해야함. 일반 네이버 메일/지메일 계정을 통해 대량 발송시, 스팸우려해 발송 거부 가능성이 있음
- 대량 발송 서비스 : AWS SES (Simple Email Service) [:link:](https://aws.amazon.com/ko/ses/)
- SMTP (Simple Mail Transfer Protocol)이나 특정 서비스의 API를 활용
